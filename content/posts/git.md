## Local config
```
>> git config --global user.email "email@example.com"
```

## Reset last commit
```
>> git reset HEAD~1
```

## Status of all files
```
>> git status -u
```

## Update last commit
```
>> git commit --amend
>> git push origin <branch> --force
```
